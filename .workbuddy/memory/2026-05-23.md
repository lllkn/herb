## 修复行走层碰撞不生效

**根因：** `this.tiledMap.layers[i].data` 是 Phaser 内部的 2D Tile对象数组（非空=Tile对象，空=null），不是原始 GID 数值。`data[i] === 0` 永远为 false，碰撞层全为 0。
**修复：** 改为从 `this.cache.json.get('gameMap')` 读取原始 JSON 的 `layers[].data`（1D GID 数组，0=空）
