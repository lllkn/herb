## 二、背包系统接口

### 2.1 数据层 — `src/gameState.js`

| 方法 | 签名 | 说明 |
|------|------|------|
| `addHerbToBackpack` | `(herbId: string): boolean` | 添加草药到背包，数量+1；**首次采集自动解锁图鉴**；返回 `true` 表示新解锁 |
| `getHerbCount` | `(herbId: string): number` | 获取指定草药数量 |
| `serialize` | `(): Object` | 序列化整个 state 为 JSON（含背包数据） |
| `save` | `(): void` | 保存到 localStorage (`'baicao_game_state'`) |

**核心状态字段：**
```javascript
state.backpack = { 'gancao': 5, 'danggui': 2, ... }  // herbId → count
state.inventory = { 'bag_luggage': 1, ... }            // itemId → count
state.collectedCount = 0                                // 总采集次数
state.unlockedHerbs = ['gancao', ...]                   // 已解锁图鉴列表
```

### 2.2 UI层 — `src/ui.js`

| 方法 | 签名 | 说明 |
|------|------|------|
| `updateBackpackUI` | `(): void` | **背包主入口**：读取 `gameStateManager.state`，按 `currentCategory` 过滤，渲染 5×4 格子网格、分类标签、详情面板 |
| `createItemCell` | `(item: Object): HTMLElement` | 创建单个物品格子 DOM，绑定点击 |
| `selectItem` | `(itemId: string, type: 'herb'\|'item'): void` | 选中物品，高亮格子 + 显示右侧详情 |
| `showDetailPanel` | `(itemId: string, type: 'herb'\|'item'): void` | 在 `#backpack-detail` 渲染详情（名称、描述、品质、数量） |
| `updateBackpackHeader` | `(state: Object): void` | 更新头部：负重、节气、背包名称 |
| `bindCategoryEvents` | `(): void` | 绑定 9 个分类标签的点击筛选事件 |
| `openModal` | `(modalId: string): void` | 打开指定弹窗（如 `'backpack-modal'`） |

**内部属性：**
```javascript
selectedHerbId = null        // 当前选中物品ID
selectedItemType = 'herb'    // 物品类型：'herb' | 'item'
currentCategory = 'all'      // 当前分类筛选
TOTAL_SLOTS = 20             // 背包格子数（5列×4行）
```

### 2.3 剧情层 — `src/IntroScene.js`

| 方法 | 签名 | 说明 |
|------|------|------|
| `_tutorialOpenBackpack` | `(step: Object): void` | 教程步骤：刷新背包UI → 打开背包弹窗 → 监听关闭后继续剧情 |
| `_tutorialHighlightHerbInfo` | `(step: Object): void` | 教程步骤：高亮背包中某草药的详情信息 |

**调用链路（教程中打开背包）：**
```
step.type === 'tutorial' && step.tutorialAction === 'open_backpack'
  → uiManager.updateBackpackUI()
  → uiManager.openModal('backpack-modal')
  → _setupModalCloseCallback()  // 关闭后回到剧情继续
```

### 2.4 采集触发 — `src/GameScene.js`

| 方法 | 签名 | 说明 |
|------|------|------|
| `checkCollection` | `(): void` | 每帧检测：遍历 `this.herbs[]`，计算玩家距离，近时显示提示 + E键响应 |
| `collect` | `(herb: Object): void` | 采集草药：标记 `collected=true`，隐藏精灵 → `addHerbToBackpack()` → 刷新背包+图鉴+任务进度 |

**调用链路：**
```
GameScene.update()
  → checkCollection()           // 距离检测 + E键 JustDown
  → collect(nearHerb)
    → gameStateManager.addHerbToBackpack(id)
    → uiManager.updateBackpackUI()
    → uiManager.updateHerbGuideUI()
    → uiManager.updateTaskProgress()
```

### 3.4 剧情解锁 — `src/IntroScene.js`

| 方法 | 签名 | 说明 |
|------|------|------|
| `_unlockHerb` | `(herbId: string): void` | 剧情中解锁指定草药图鉴：调用 `addHerbToBackpack()` → 同步 `gameState.unlockedHerbs` → 刷新UI |
| `_tutorialOpenHerbPedia` | `(step: Object): void` | 教程步骤：打开图鉴弹窗（`openModal('herb-guide-modal')`） |

**`_unlockHerb` 完整链路：**
```javascript
_unlockHerb(herbId):
  → window.gameStateManager.addHerbToBackpack(herbId)    // 数据层解锁
  → this.gameState.unlockedHerbs.push(herbId)             // IntroScene 本地同步
  → window.uiManager.updateBackpackUI()                   // 刷新背包
  → window.uiManager.updateHerbGuideUI()                  // 刷新图鉴
```

## 四、第一章剧情触发接口

### 4.1 章节启动 — `src/BootScene.js`

| 变量/方法 | 说明 |
|------|------|
| `window._loadChapter` | 全局章节标记：`0` = 序章，`1` = 第一章 |
| `window._chapter1Data` | 第一章剧情数据缓存（`story_chapter1.json`） |
| `BootScene.create()` | 启动决策：优先 `window._loadChapter` ≥ 1（调试面板） → 次查 `gameStateManager.state.currentChapter`（存档） → 否则默认序章 |

**决策逻辑：**
```javascript
if (window._loadChapter >= 1 && window._chapter1Data) {
    // 调试指定 → 第一章
} else if (currentChapter >= 1 && window._chapter1Data) {
    window._loadChapter = 1;  // 存档记录 → 第一章
} else {
    window._loadChapter = 0;  // 默认 → 序章
}
```

### 4.2 剧情引擎 — `src/IntroScene.js`

#### 场景管理

| 方法 | 签名 | 说明 |
|------|------|------|
| `create` | `(): void` | 场景入口：根据 `window._loadChapter` 加载序章/第一章数据，调用 `_initScene()` |
| `_initScene` | `(): void` | 初始化子系统 → 创建UI → 设置输入 → 启动DebugManager → `_startScene(0)` |
| `_startScene` | `(sceneIndex: number): void` | 启动指定索引的场景，从 `prologueData.scenes[sceneIndex]` 读取 steps |
| `_executeStep` | `(): void` | **核心步骤执行器**：根据 `step.type` 分流到对应处理方法 |

#### Step 类型分发

| step.type | 处理方法 | 说明 |
|------|------|------|
| `'narration'` | 内联渲染 | 居中旁白文字 |
| `'dialogue'` | `_showDialogue(step)` | 对话框文字 + 打字机 + 头像/立绘 |
| `'choice'` | `_showChoice(step)` | 分支选项（调用 ChoiceSystem） |
| `'tutorial'` | `_showTutorial(step)` | 教程步骤（采集/背包/图鉴交互） |
| `'cg'` | `cgDisplay.show(step)` | CG图片/信件演出 |
| `'wait'` | 时间延迟 | 等待指定 ms |
| `'end'` 或最后一步 | `_endPrologue(step)` | 章节结束 → 存档 → 跳转 GameScene |
| `'background'` | 内联渲染 | 切换背景 |
| `'bgm'` | 内联渲染 | 播放背景音乐 |
| `'effect'` | 内联渲染 | 特效（淡入/淡出/震动等） |
| `'unlock_herb'` | `_unlockHerb(step.herbId)` | 剧情中解锁图鉴 |

#### 关键方法详表

| 方法 | 签名 | 说明 |
|------|------|------|
| `_showDialogue` | `(step: Object): void` | 显示对话。设置 speakerName/avatar/image/portrait → 打字机逐字输出 content → 同时处理 `step.setFlag` 标记 |
| `_showChoice` | `(step: Object): void` | 显示选项列表。创建 ChoiceSystem 实例 → `choiceSystem.show(step.options)` → 选择后 `_onChoiceSelected()` |
| `_onChoiceSelected` | `(option: Object, index: number): void` | 选项回调。清除 `this.currentSteps`（跳过当前场景剩余步骤）→ 调用 `_startScene(option.next)` 跳转 |
| `_showTutorial` | `(step: Object): void` | 教程步骤。根据 `step.tutorialAction` 分发：collect / open_backpack / open_herb_pedia / highlight 等 |
| `_setFlag` | `(flag: string, value: any): void` | 设置剧情标记到 `this.gameState.flags[flag]` |
| `_endPrologue` | `(step: Object): void` | 章节结束。第一章：`gameStateManager.state.currentChapter=1` → save → `scene.start('GameScene')`；序章：初始化属性 → save → GameScene |
| `_saveGameState` | `(): void` | 保存 `this.gameState` 到 localStorage `'baicao_game_state'` |

## 五、跨系统联动链路

### 5.1 采集 → 背包 → 图鉴（完整链路）

```
GameScene.checkCollection()   // 距离检测
  → GameScene.collect(herb)
    → gameStateManager.addHerbToBackpack(id)
      → state.backpack[id]++          // 背包+1
      → state.collectedCount++        // 总采集+1
      → state.unlockedHerbs.push(id)  // 首次采集自动解锁图鉴
      → 返回 isNewUnlock = true
    → uiManager.showCollectSuccess()   // 飘字提示
    → uiManager.updateBackpackUI()     // 刷新背包
    → uiManager.updateHerbGuideUI()    // 刷新图鉴
    → uiManager.updateTaskProgress()   // 刷新任务
```

### 5.2 剧情 → 背包 → 图鉴（教程联动）

```
IntroScene._executeStep()
  → step.type === 'tutorial'
    → tutorialAction === 'play_collect_animation'
      → gameStateManager.addHerbToBackpack(herbId)
      → uiManager.updateHerbGuideUI()
    → tutorialAction === 'open_backpack'
      → uiManager.updateBackpackUI()
      → uiManager.openModal('backpack-modal')
      → _setupModalCloseCallback() → 回到剧情
    → tutorialAction === 'open_herb_pedia'
      → uiManager.openModal('herb-guide-modal')
    → tutorialAction === 'highlight_herb_info'
      → _tutorialHighlightHerbInfo()
```

### 5.3 剧情解锁图鉴（`_unlockHerb` 联动）

```
IntroScene._executeStep()
  → step.type === 'unlock_herb'
    → _unlockHerb(herbId)
      → gameStateManager.addHerbToBackpack(herbId)  // 数据层
      → this.gameState.unlockedHerbs.push(herbId)    // IntroScene 本地同步
      → uiManager.updateBackpackUI()
      → uiManager.updateHerbGuideUI()
```

### 5.4 章节结束 → GameScene 跳转

```
IntroScene._endPrologue()
  ├─ 第一章结束:
  │   → gameStateManager.state.currentChapter = 1
  │   → gameStateManager.save()
  │   → _showHTMLUI()                    // 恢复地图UI
  │   → scene.start('GameScene')
  │
  └─ 序章结束:
      → _initPrologueAttributes()        // 初始化《本草情籍》
      → _saveGameState()                  // localStorage
      → scene.start('GameScene')
```

### 5.5 F3 调试触发章节

```
用户按 F3 → DebugManager panel 显示
  ├─ 序章下拉 → 选择 S01-S09 → forceChapter=0 → _startScene(idx)
  └─ 第一章下拉 → 选择 EVT_xxx → forceChapter=1 → _startScene(idx)
    (通过 EVENT_TO_SCENE 映射到 C01-C16)
```

### 6.4 存档 key 汇总

| localStorage Key | 写入者 | 内容 |
|------|------|------|
| `'baicaoxing_has_played'` | GameScene | `'true'` — 是否首次游戏 |
| `'gameSettings'` | Settings (main.js) | UI 设置（音量、键位等） |
| `'baicao_game_state'` | `gameStateManager.save()` / `IntroScene._saveGameState()` | 完整游戏状态 JSON |
