## 二、设计原则：地图叙事规范

### 2.1 三种事件触发方式

| 触发方式 | 使用场景 | 实现字段 |
|---------|---------|---------|
| **区域进入**（Zone Enter） | 进入某个矩形区域自动触发 | `trigger: "zone"` + `zoneId` |
| **NPC 交互**（E 键对话） | 靠近 NPC 后按 E 键 | `trigger: "npc"` + `npcId` |
| **物件交互**（E 键检查） | 靠近地物（水井/晒药台/蛊根草）后按 E | `trigger: "object"` + `objectId` |

### 2.2 触发频率控制

```javascript
// 每个事件最多触发一次（除草药外）
// story_events_fired = new Set()  存入 gameState
// 触发前检查：if (gameState.storyEventsFired.has(eventId)) return;
// 触发后：gameState.storyEventsFired.add(eventId)
```

### 2.3 剧情对话在地图上的呈现

- **小对话**（1-3行）：Phaser 内置气泡框，不打断移动，角色头顶显示
- **重要对话**（4行以上）：调用 IntroScene 式的底部对话框（DOM overlay），**不切换场景**，对话期间玩家移动禁用
- **小游戏**：全屏覆盖 HTML panel，完成后自动恢复地图控制

### 2.4 任务追踪 HUD

左侧任务面板实时更新，分两层：
- **主线任务**（橙色竖条）：当前章节主要目标
- **探索提示**（灰色竖条）：「附近有可交互对象」「青苗感知到草药」

## 六、触发器技术规范

### 6.1 事件数据结构

所有事件存储在 `src/data/map_events_plains.json` 和 `src/data/map_events_valley.json`。

```javascript
// GameScene.js 初始化时加载
this.load.json('events_plains', 'src/data/map_events_plains.json');
this.load.json('events_valley', 'src/data/map_events_valley.json');

// 运行时事件系统
class StoryEventManager {
    constructor(scene) {
        this.scene = scene;
        this.events = {};        // eventId → event config
        this.fired = new Set();  // 已触发事件集合
    }

    register(eventsArray) {
        eventsArray.forEach(e => this.events[e.eventId] = e);
    }

    // 每帧调用（仅检查 zone 类型）
    checkZones(playerX, playerY) {
        Object.values(this.events).forEach(evt => {
            if (evt.trigger !== 'zone') return;
            if (evt.once && this.fired.has(evt.eventId)) return;
            const zone = this.scene.zoneMap[evt.zoneId];
            if (zone && Phaser.Geom.Rectangle.Contains(zone, playerX, playerY)) {
                this.fire(evt.eventId);
            }
        });
    }

    // E键交互时调用
    checkInteraction(targetId, targetType) {
        Object.values(this.events).forEach(evt => {
            if (evt.trigger !== targetType) return;
            const id = targetType === 'npc' ? evt.npcId : evt.objectId;
            if (id !== targetId) return;
            this.fire(evt.eventId);
        });
    }

    fire(eventId) {
        const evt = this.events[eventId];
        if (!evt) return;
        if (evt.once && this.fired.has(eventId)) return;

        // 检查前置条件
        if (evt.condition && !this.checkCondition(evt)) return;

        this.fired.add(eventId);
        gameState.storyEventsFired.add(eventId);  // 持久化

        // 触发对话
        window.storyDialogSystem.playSteps(evt.steps, evt.dialogType);

        // 标记地点完成
        if (evt.locComplete) {
            gameState.completedLocations.add(evt.locComplete);
            this.checkVillageComplete();
        }
    }

    checkVillageComplete() {
        const required = ['loc_herb_garden','loc_well','loc_drying_platform','loc_empty_shop','loc_zhang_home'];
        if (required.every(loc => gameState.completedLocations.has(loc))) {
            this.fire('EVT_VILLAGE_COMPLETE');
        }
    }
}
```

### 6.2 对话展示方式

| dialogType | 表现形式 | 玩家移动 |
|-----------|---------|---------|
| `bubble`  | NPC头顶浮动气泡，自动消失 | 不限制 |
| `overlay` | 底部全宽对话框（DOM），支持立绘 | **禁止** |

```javascript
// storyDialogSystem.playSteps() 内部
if (dialogType === 'bubble') {
    // Phaser Text 气泡，3s 后自动隐藏，不暂停游戏
    this.showBubble(npcX, npcY, steps[0].text);
} else {
    // DOM overlay 对话框，暂停玩家输入
    this.scene.player.body.setVelocity(0, 0);
    this.scene.input.keyboard.enabled = false;
    this.showOverlayDialog(steps, () => {
        this.scene.input.keyboard.enabled = true;
    });
}
```
