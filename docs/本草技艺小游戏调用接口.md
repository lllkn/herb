# 本草技艺小游戏调用接口

已拆分的三个小游戏：

| 剧情 step.type | 玩法 | 结果状态键 |
| --- | --- | --- |
| `minigame_licorice_harvest` | 甘草采挖 | `__herbMinigame_licorice_Result` |
| `minigame_chrysanthemum_pick` | 菊花采摘 | `__herbMinigame_chrysanthemum_Result` |
| `minigame_herb_grind` | 研磨药粉 | `__herbMinigame_grind_Result` |

三个小游戏完成后都会继续推进当前剧情，并把最近一次结果写入：

```js
window.gameStateManager.state.__herbMinigameResult
```

## 剧情 JSON 示例

```json
{
  "id": "CXX_M01",
  "type": "minigame_licorice_harvest",
  "rewardOnCorrect": { "attr": "crafting", "delta": 2 },
  "resultFlag": "__storyLicoriceResult"
}
```

```json
{
  "id": "CXX_M02",
  "type": "minigame_chrysanthemum_pick",
  "minigameOptions": { "timer": 25, "respawnDelay": 45 },
  "rewardOnCorrect": { "attr": "herbalism", "delta": 2 }
}
```

```json
{
  "id": "CXX_M03",
  "type": "minigame_herb_grind",
  "rewardOnCorrect": { "attr": "crafting", "delta": 3 }
}
```

## 直接 JS 调用

```js
window.HerbMinigameAPI.start('licorice').then(result => {
  console.log(result);
});

window.HerbMinigameAPI.start('chrysanthemum', { timer: 25, respawnDelay: 45 });
window.HerbMinigameAPI.start('grind');
```

`result` 基础字段：

```js
{
  id: 'licorice',
  success: true,
  timestamp: 1710000000000
}
```

各小游戏会额外返回自己的分数或品质字段，例如 `score`、`picked`、`quality`。
