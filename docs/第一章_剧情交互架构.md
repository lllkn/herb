#### C01 · 出生区搭车过场 🚗

```
玩家进入地图"出生区" → 触发 zone_map_entry → IntroScene 启动
```

| 步骤ID | 类型 | 内容 | 交互方式 | 后续 |
|--------|------|------|----------|------|
| C01_001 | narration | "山路蜿蜒，赶车的老伯在翠竹村岔路口将你放下..." | 自动播放(2.5s) | 下一步 |
| C01_002 | dialogue | 青苗："终于到了！主人，这里草药的气息好浓！" | 点击继续 | 下一步 |
| C01_003 | **tutorial** | **操作提示："WASD移动·靠近草药按E采集·靠近NPC按E对话"** | **点击跳过** | **场景结束→C02** |

> **接口说明**: `C01_003` 的 `tutorialAction=movement_hint` 在第一章模式下变为"显示提示文字+点击即完成"，不阻塞游戏流程。

**触发接口**:
```json
{
  "_trigger": {
    "type": "zone",
    "zoneId": "zone_map_entry",
    "once": true,
    "dialogType": "overlay",
    "eventId": "EVT_PLAINS_ENTRY"
  }
}
```

│  → 无论选哪个都汇聚到 C03_002                       │
└─────────────────────────────────────────────────────┘

**触发接口**:
```json
{ "type": "npc", "npcId": "woodcutter_npc", "once": false, "eventId": "EVT_WOODCUTTER" }
```
> `once: false` = 可重复对话（每次遇到都可触发）

## 四、交互类型汇总

### 支持的所有步骤类型

| 类型标识 | 中文名称 | 交互方式 | 典型用途 |
|----------|----------|----------|----------|
| `narration` | 旁白 | 自动播放 | 氛围渲染、转场 |
| `dialogue` | 对话 | 点击继续 | 角色交流、信息传递 |
| `choice` | 选择 | **多选一** | 分支决策、角色扮演 |
| `tutorial` | 教程提示 | **点击跳过** | 操作指引、系统教学 |
| `minigame_herb_id` | 辨药小游戏 | **选择题** | 知识检测、属性变化 |
| `minigame_process` | 炮制小游戏 | **操作演示** | 技能学习 |
| `minigame_diagnosis` | 四诊断小游戏 | **4轮选择** | 核心玩法、专业模拟 |
| `minigame_prescription` | 处方小游戏 | **背包选材** | 核心玩法、策略性 |
| `cg` | CG演出 | **自动播放** | 高潮时刻视觉冲击 |
| `reward` | 奖励发放 | **自动** | 成就感、资源积累 |
| `title_card` | 标题卡 | **自动展示** | 章节分隔 |
| `end` | 结束标记 | **自动** | 切换场景 |

### 触发方式分类

| 触发类型 | JSON字段 | 说明 | 示例 |
|----------|----------|------|------|
| **zone** | `"type": "zone"` | 区域进入触发 | C01/C02/C07/C15/C16 |
| **npc** | `"type": "npc"` | 与NPC对话触发 | C03/C05/C06/C09/C10/C13 |
| **object** | `"type": "object"` | 检查物品触发 | C04/C11/C12 |
| **condition** | `"type": "condition"` | 满足条件触发 | **C14**(全部地点完成) |
